# Security

Do not open public issues for security vulnerabilities.

Report privately via GitHub Security Advisories on
[DCraft-Labs/dcraft-fusion](https://github.com/DCraft-Labs/dcraft-fusion/security/advisories/new).

The canonical policy is:

**[SECURITY.md](https://github.com/DCraft-Labs/dcraft-fusion/blob/main/SECURITY.md)**

Include affected component (control plane, web, CDC engine, etc.), reproduction steps, and impact when you can.
