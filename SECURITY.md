# Security Policy

## Supported versions

Security fixes are prioritized for the current `main` branch of
[DCraft-Labs/dcraft-fusion](https://github.com/DCraft-Labs/dcraft-fusion).

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities privately via
[GitHub Security Advisories](https://github.com/DCraft-Labs/dcraft-fusion/security/advisories/new)
on the `DCraft-Labs/dcraft-fusion` repository.

Include as much detail as you can:

- Affected component or path (for example control plane, web app, CDC engine)
- Steps to reproduce
- Impact assessment
- Any suggested remediation

We will acknowledge valid reports and work with you on coordinated disclosure.
