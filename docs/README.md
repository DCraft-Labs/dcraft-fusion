# DCraft Fusion Docs

This folder contains the project source of truth.

- `product/`: canonical product context and positioning.
- `architecture/`: frozen V0 architecture, technical architecture, deep dives, and connector/Kubernetes patterns.
- `roadmap/`: V0 to V1 feature roadmap and phase planning.
- `adr/`: architecture decisions created during implementation.
- `api/`: API design notes and generated docs.
- `runbooks/`: operational procedures.
