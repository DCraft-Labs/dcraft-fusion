# Open core

DCraft Fusion is an **open-core** product from DCraft Labs.

## Community (Apache 2.0)

The open-source Community edition includes:

- Control plane kernel and operator UX
- **Fusion CDC engine** (included in v1 — not an Enterprise upsell)
- Simple password / JWT authentication for labs and early deployments
- Helm charts and images via GHCR (`oci://ghcr.io/dcraft-labs/charts/...`)
- Engine-agnostic coordination, metadata, policy, run state, and audit foundations
- Community support via GitHub Issues and Discussions

## Enterprise (gated)

Enterprise builds on Community and may include:

- SSO / SAML / OIDC and SCIM provisioning
- BYOC deployment patterns
- SLA and priority support
- Managed / hosted control plane options
- Commercial licensing terms

Enterprise features are **not** required to run Community. CDC remains in Community.
