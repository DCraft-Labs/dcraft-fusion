# AI Project Context

Use `../PROJECT_CONTEXT.md` and `../docs/product/DCraft_Fusion_PROJECT_CONTEXT_FOR_CODEX_V2_EXPANDED.md` as the primary context.

DCraft Fusion is an AI-assisted, engine-agnostic data platform control plane. The AI assistant must preserve the core product boundary: Fusion coordinates external tools; it does not become the default warehouse, lake, orchestration engine, BI engine, or compute engine.

Current build priority:

1. Control plane kernel.
2. UX shell.
3. Connection registry and metadata discovery.
4. One adapter integration.
5. Run/audit tracking.
6. AI recommendations as drafts with approval.
