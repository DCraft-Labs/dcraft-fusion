# Contributing

Contributions are welcome via fork and pull request against
[DCraft-Labs/dcraft-fusion](https://github.com/DCraft-Labs/dcraft-fusion).

The canonical contributor guide is the repository root document:

**[CONTRIBUTING.md](https://github.com/DCraft-Labs/dcraft-fusion/blob/main/CONTRIBUTING.md)**

It covers:

- Issue and discussion channels
- Fork / PR workflow
- Developer Certificate of Origin (DCO) sign-off (`git commit -s`)
- Local `npm test`, `typecheck`, and `build`
- Compose stack for end-to-end checks

Also read the [Code of Conduct](https://github.com/DCraft-Labs/dcraft-fusion/blob/main/CODE_OF_CONDUCT.md) and [Security](/security) policy.
