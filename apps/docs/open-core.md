# Open core

DCraft Fusion is an **open-core** product from DCraft Labs.

## Community

- **Control plane** source: Apache 2.0
- **CDC**: run via **public GHCR images** + public Helm chart (connector/worker source is private)
- Simple password / JWT authentication
- Community support via GitHub Issues and Discussions

## Enterprise (gated)

- SSO / SAML / OIDC and SCIM
- BYOC, SLA, managed offerings
- Commercial support

Enterprise is **not** required to run Community CDC images.
